# Proyecto-Final

El presente proyecto tiene como finalidad desarrollar una versión digital del juego tradicional “Piedra, Papel o Tijera”, aplicando los conocimientos adquiridos durante las cuatro unidades de la asignatura.

# Objetivo del Programa

Desarrollar un sistema interactivo en Python que permita:

Simular el juego Piedra, Papel o Tijera.

Comparar elecciones entre usuario y computadora.

Determinar automáticamente el ganador.

Aplicar estructuras de datos y programación funcional.

# Funcionalidades del Código

Generación aleatoria de la jugada de la computadora.

Validación de entrada del usuario.

Determinación automática del ganador.

Uso de funciones para organizar el programa.

Posibilidad de jugar múltiples rondas.
